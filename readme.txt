Custom post meta field and meta box to set a date line for a news story.

== Description ==

Date Line provides the ability to set a date line in a news story.

== Installation ==

1. Upload the `date-line` directory to your site's `plugins` directory
2. Activate the plugin through the `Plugins` menu in WordPress

== Frequently Asked Questions ==

= Does the plugin do any automatic formatting? =
No, the plugin will display the text exactly as you enter it.
